# Scientific Paper: An Introduction to General Relativity

This repository contains the LaTeX source and a PDF for the paper 
*"An Introduction to General Relativity: From Principles to Predictions"* 
by Daniel Alejandro Hinojosa.

## Overview
The paper is a thorough introduction to Einstein's theory of general relativity, written for an undergraduate-level audience. It moves from the historical context of Newtonian gravity to the core principles and experimental confirmations of general relativity, such as the precession of Mercury's perihelion and the need for relativistic corrections in GPS satellites.

## Repository Contents
- `main-paper.tex`: The LaTeX source code of the paper.
- `main-paper.pdf`: The final paper in PDF format.
- `main-presentation.pptx`: A PowerPoint presentation summarizing the paper's key findings.
- `attachments/`: A folder containing image files used in the paper.

## About the Author
This paper was written by Daniel Alejandro Hinojosa as part of a Scientific Methodology course, demonstrating strong skills in research, writing, and technical communication.
